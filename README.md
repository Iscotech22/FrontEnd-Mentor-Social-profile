# 🌐 Social Links Profile

A clean and responsive social links profile card built with **HTML** and **CSS**.

This project displays a user profile with an avatar, location, short bio, and social media links in a modern card layout.

---

## 📌 Overview

The application contains:

* 👤 Profile image/avatar
* 📝 User information
* 📍 Location display
* 💬 Short bio/quote
* 🔗 Social media links

The layout is centered on the page and designed with a minimal dark-themed interface.

---

## 🚀 Features

* Responsive card layout
* Semantic HTML structure
* Organized content sections
* Social media link buttons
* Accessible image alt text
* Clean UI design
